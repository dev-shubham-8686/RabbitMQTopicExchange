Action,Command
Start Services (Background), docker-compose up -d
Check Health/Status, docker-compose ps
View Live Logs, docker-compose logs -f
Stop Services, docker-compose stop
Stop and Remove Containers, docker-compose down
Reset Everything (Wipe Data), docker-compose down -v



Action,Command
List Running Containers,docker ps
List All (even stopped),docker ps -a
Stop RabbitMQ,docker stop rabbitmq
Start RabbitMQ again,docker start rabbitmq
Check RabbitMQ Logs,docker logs -f rabbitmq
Delete RabbitMQ,docker rm -f rabbitmq
Check Redis Health,docker exec -it redis redis-cli ping
